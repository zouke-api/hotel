登录

1. /auth/is-login

描述：检测当前用户是否处于登录状态

request：

{ "token": "fdON29dizu-yowf44u7gpMLlxGnjnf4P" }

response：

{ "code": 0, "login": true, "uid": 10000 }

说明：请求token是当前用户身份标识，响应login表示token是否有效，无其它特殊code值。
