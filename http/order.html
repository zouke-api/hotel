订单/支付

1. /ht-app/order/commit

描述：提交订单

request：

{
    "contactTel": "18888888888999", # 电话号码
    "invoice": {                    # 发票
        "need": true,               # 是否需要发票
        "addr": "公司地址",          # 公司地址
        "country": "other",         # 国家
        "number": "税号",
        "title": "抬头",
        "weix": "联系微信"
    },
    "persons": [
        [ 
            {
                "first": "名",
                "last": "姓"
            }, 
            {
                "first": "名",
                "last": "姓"
            } 
        ]
    ],
    "remark": [
        "big","smokeFree" # big尽量安排大床 twoSingle尽量安排双床 smokeFree尽量安排无烟房 smoke尽量安排吸烟房 baby需要婴儿床 airConditioning需要空调
    ],# 用户备注
    "sessionObj": { # 详情接口的选择房型数据中对应的sessionObj字段
        ...
    }
}

response：

{
    "code": 0,
    "order_id": 100000
}

说明：请求参数sessionObj是酒店详情中价格列表结构中对应的sessionObj
